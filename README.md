# Akıllı Görev Yöneticisi
### Bu proje, terminal üzerinden çalışan, fonksiyonel programlama prensiplerine uygun olarak geliştirilmiş bir görev (To-Do) yönetim uygulamasıdır. Kullanıcıların görev eklemesine, listelemesine ve mevcut iş yükünü analiz etmesine olanak tanır.

## Ne Yapıyor?
* Görev Ekleme: Kullanıcıdan alınan görevleri bir liste veri yapısında saklar.

* Dinamik Listeleme: Mevcut görevleri numaralandırarak (enumerate) kullanıcıya sunar.

* İş Yükü Analizi: Görev sayısına göre kullanıcının yoğunluk durumunu (Başlangıç veya Yoğun) raporlar.

* Sürekli Çalışma: Kullanıcı "Çıkış" diyene kadar menü yapısını aktif tutar.

## Hangi Konular Var?
* Fonksiyonel Programlama: Kodun tekrarını önlemek için def ile tanımlanmış modüler yapılar.

* Döngüler: Uygulamanın kapanmadan çalışmasını sağlayan while True döngüsü.

* Gelişmiş String Formatlama: f-string kullanımı ile daha okunaklı çıktılar.

* Hata Yönetimi Mantığı: Liste boş olduğunda kullanıcıyı uyaran kontrol mekanizmaları.
  
## Nasıl Çalıştırılır?
* Bilgisayarınızda Python yüklü olduğundan emin olun.

* Terminali açın ve proje dizinine gidin.

* Aşağıdaki komutla uygulamayı başlatın:

```Bash
python gorev_yoneticisi.py

```
* Menüdeki numaraları (1, 2, 3, 4) kullanarak işlemleri gerçekleştirin.
